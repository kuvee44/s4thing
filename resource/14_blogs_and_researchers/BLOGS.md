## RSS / Feed Tips

Most researcher blogs support RSS. Recommended aggregator setup:
- Add tiraniddo.dev, itm4n.github.io, googleprojectzero.blogspot.com as priority feeds
- Monitor Project Zero issue tracker: https://bugs.chromium.org/p/project-zero/issues/list
- Follow MSRC Security Update Guide for patch Tuesday context: https://msrc.microsoft.com/update-guide/
